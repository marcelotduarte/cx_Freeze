
Welcome to cx_Freeze's documentation!
=====================================

**cx_Freeze** creates standalone executables from Python scripts, with the same
performance, is cross-platform and should work on any platform that Python_
itself works on.

.. list-table::
   :header-rows: 1
   :widths: 300 300 200

   * - cx_Freeze version
     - Python version
     - Status
   * - cx_Freeze 6.4 to 6.8
     - Python 3.6 to 3.9
     - supported
   * - cx_Freeze 6.2, 6.3
     - Python 3.5.2 to 3.8
     - unsupported
   * - cx_Freeze 6.1
     - Python 3.5.0 to 3.8
     - unsupported
   * - cx_Freeze 6.0
     - Python 3.5.0 to 3.7
     - unsupported
   * - cx_Freeze 5.1.1
     - Python 2.7
     - unsupported
