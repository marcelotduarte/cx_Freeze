About cx_Freeze

cx_Freeze creates standalone executables from Python scripts, with the same performance, is cross-platform, and should work on any platform that Python itself works on. It supports Python 3.9 up to 3.13. If you need support for older Python check the documentation.

Installation

Version 8.1, released April, 02, 2025

In a virtual environment, install by issuing the command: 

pip install --upgrade cx_Freeze
